Note that a player's strength remains constant throughout a tournament. For example, PW was equally strong in Game 1 and in Game 2. A player neither gets stronger nor weaker between games.

However, each player can choose to be lazy in any given game. If a player decides to be lazy, then that player doesn't pull with all their strength. For example, it's possible that PW pulled with all his strength in Game 1 but decided to be lazy in Game 2.

A team's strength is determined by the sum of the strengths of its players. The team with the greater overall strength wins the game.
